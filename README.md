# UI-UX-Experiment5

## Aim:
To analyze UX project scenarios, identify user problems, select suitable research methods, and create user personas for user-centered design.

## Algorithm:
1.Select two UX project scenarios.

2.Identify user pain points in each scenario.

3.Write UX problem statements using the given template.

4.Choose appropriate research methods.

5.Justify the selected methods.

6.Create user personas with demographics, goals, behaviors, and pain points.

7.Analyze findings and recommend improvements.

## Output:
[UI & UX epx-5.pdf](https://github.com/user-attachments/files/28542670/UI.UX.epx-5.pdf)


## Result:
Two UX project scenarios were analyzed. User pain points were identified, problem statements were created, appropriate research methods were selected and justified, and user personas were developed to support user-centered design decisions.
